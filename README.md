# Röstigraben bei GA und Halbtax


## Artikel

Die Daten und Methoden in diesem Repository beziehen sich auf den Artikel [Röstigraben bei GA und Halbtax](http://www.srf.ch/data), publiziert am 21. September 2017.

### Vorprozessierung und Analyse

Der Ordner `analysis` beinhaltet die Prozessierungsschritte (inkl. explorativer Analyse) in R. Der ganze Prozess steht ebenfalls [unter diesem Link](http://srfdata.github.io/2017-09-sbb-ga-halbtax/) als HTML-File zur Verfügung.
